# Codigo-Banco-de-Dados


# 📚 Código Banco de Dados

Repositório criado para armazenar atividades, exercícios e projetos desenvolvidos durante os estudos de Banco de Dados no curso de Análise e Desenvolvimento de Sistemas — SENAI CIMATEC.

---

# 🚀 Tecnologias Utilizadas

* SQL
* MySQL
* Modelagem de Banco de Dados
* Git e GitHub

---

# 📂 Atividades do Repositório

## 🛒 ECOMERCIAL.sql

Sistema de banco de dados para um e-commerce.

### Conteúdos praticados:

* Criação de tabelas
* Relacionamentos
* Inserção de dados
* Consultas SQL

---

## 🍱 Sistema Marmitas.sql

Projeto de banco de dados para gerenciamento de pedidos de marmitas.

### Funcionalidades:

* Cadastro de clientes
* Cadastro de pedidos
* Controle de produtos

---

## 🛍 lojaVirtual.sql

Estrutura de banco de dados de uma loja virtual.

### Conceitos aplicados:

* Chaves primárias
* Chaves estrangeiras
* Relacionamentos entre tabelas

---

## 💰 Atv Vendas.sql

Atividade prática envolvendo sistema de vendas.

### Objetivos:

* Treinar comandos SQL
* Realizar consultas
* Trabalhar com registros de vendas

---

## 📦 atvDDL.sql

Exercícios utilizando comandos DDL.

### Comandos utilizados:

* CREATE
* ALTER
* DROP

---
